## 🚀 About This Portfolio

Welcome to my digital playground! This portfolio is designed not just to display my resume, but to provide a **premium user experience** built with a custom deep black and vivid red color palette (`#121212` & `#b74b4b`). It features buttery-smooth scroll animations, dynamic filtering, and a hidden interactive OS-like terminal!

## ✨ Key Features

- **🎨 Custom Theming:** A highly customized dark mode UI that feels sleek and hacker-like.
- **📱 Fully Responsive:** Carefully optimized layouts that look stunning on desktops, tablets, and phones.
- **🪄 Micro-Animations:** Driven by CSS and the **AOS (Animate on Scroll)** library to bring the page to life.
- **🏆 Dynamic Achievements Page:** A dedicated, filterable grid displaying certificates and milestones.
- **💻 Interactive Linux Terminal:** A built-in command-line interface that visitors can interact with (see below).

## 💻 The Hidden Terminal

One of the standout features of this portfolio is the **Custom Terminal Widget**. Click the terminal icon `<i class="fas fa-terminal"></i>` on the website to open a fully simulated OS!

**Try these commands:**
- `help` — List all available commands.
- `ls` — View the virtual file system.
- `blog` / `read [id]` — Read my headless developer blog directly in the terminal!
- `ask [question]` — Chat with my automated AI assistant to learn about my skills.
- `play snake` — Play a fully functioning game of Snake right in the CLI.
- `quote` — Fetch a random inspirational developer quote.

***Easter Eggs:*** *Try typing `matrix`, `joke`, `coffee`, `42`, or the ultimate `sudo hire-me`...*
